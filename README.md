# Project-Dental-Clinic-Data-Simulation-Analytics

📋 **Опис проєкту:** комплексний аналітичний проєкт: від проектування архітектури даних до побудови інтерактивної BI-системи для управління стоматологічною клінікою. Проєкт демонструє вміння працювати з бізнес-вимогами, моделювати складні процеси та візуалізувати приховані інсайти.

🎯 **Мета проєкту:** продемонструвати повний цикл роботи аналітика: від консультації з власником бізнесу та збору вимог до надання готового інструменту для прийняття управлінських рішень.

**Результат узгоджено з власницею клініки — логіка звітів та структура дашборду відповідають бізнес-очікуванням.**

💻 **Технологічний стек:** Python (Pandas, NumPy), Jupyter Notebook, Tableau Desktop

🛠 **Етап 1:** Проектування та генерація даних (Python)
Оскільки реальні дані пацієнтів є конфіденційними, я розробила систему симуляції даних, яка враховує реальну бізнес-логіку та регіональний контекст. У скрипт закладено:

- графіки роботи (9:00–18:00), година обіду (плаваюча з 12:00 до 14:00), ліміти прийомів (до 5-7 осіб на лікаря, залежно від спеціалізації  та тривалості надання послуг) та автоматичний розподіл 24 днів відпустки лікарів (2х7 днів, останні розподілено протягом року).
- **регіональна специфіка (Закарпаття):** враховано сплески попиту перед святами (Великдень, Різдво, Новий рік) через приїзд мешканців з-за кордону, а також вихідні у дні великих церковних свят.
- **ієрархія даних** - послуги згруповані за категоріями та спеціалізаціями, додано маркер явки пацієнта (No-show) для аналізу потенційно втраченого доходу.

📊 **Етап 2: BI-аналітика та візуалізація (Tableau)**
На основі згенерованих даних було побудовано систему інтерактивних дашбордів для комплексного аналізу операційних показників клініки:

- **загальні показники (KPI):** моніторинг виручки (738K ₴), середнього чека (4,731 ₴) та структури пацієнтів (Нові vs Постійні) для оцінки лояльності.
- **теплові карти (Heatmaps):** візуалізація навантаження кабінетів та ефективності кожного лікаря за годинами, що дозволяє бачити реальні «вікна» у графіку.
- **аналіз доходів:** розподіл виручки за категоріями (Ортодонтія — 51%, Хірургія — 30.2%) та детальний рейтинг ТОП-послуг за прибутковістю.

🛠**Інструментарій:** використано складні обчислювальні поля (Calculated Fields) та інтерактивні фільтри (за місяцем/лікарем) для динамічної деталізації звітів.

💡**ВИСНОВКИ:**
- **Оптимізація робочого часу:** поточна завантаженість ~43% свідчить про ресурс для подвоєння записів без розширення штату. Теплова карта навантаження персоналу дозволяє точно визначити «порожні вікна» та переглянути графіки так, щоб у години пікового попиту в клініці працювало більше фахівців.
- **Структура доходів:** аналіз показує, що 51% виручки припадає на Ортодонтію. Це дозволяє побачити реальний розподіл грошей по напрямках і виявити потенціал для масштабування інших послуг (наприклад, Терапії), які мають ресурс для зростання доходності клініки.
- **Retention-контроль:** порівняння нових та постійних клієнтів допомагає вчасно помітити відтік людей. Це сигнал для власника, що діючу систему лояльності або сервіс потрібно переглянути, щоб пацієнти поверталися частіше.
